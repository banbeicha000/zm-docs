animation-timeline ​

animation-timeline CSS 属性指定用于控制 CSS 动画进度的时间轴

TIP

这是一项实验性技术

设置以下类型的时间线

  • 滚动进度时间线，通过在顶部和底部（或左侧和右侧）之间滚动可滚动元素（滚动条）来推进。滚动范围内的位置将转换为进度百分比 — 开始时为 0%，结束时为 100%。可以通过两种方式指定提供滚动进度时间线的元素
    • 命名滚动进度时间线是指使用 scroll-timeline-name 属性（或 scroll-timeline 简写属性）显式命名提供滚动进度时间线的滚动条。然后，通过将该名称指定为该元素animation-timeline属性的值，将该名称链接到要进行动画处理的元素。
    • 在匿名滚动进度时间轴中，要制作动画的元素被赋予一个 scroll（） 函数作为animation-timeline值，该函数根据传递给它的参数选择提供滚动进度时间轴的滚动条和要使用的滚动轴。
  • 视图进度时间线，根据滚动条内元素（称为主题）的可见性变化进行进度。滚动条内主体的可见性被跟踪 - 默认情况下，当主体在滚动条的一个边缘首次可见时，时间轴为 0%，当它到达另一个边缘时，时间轴为 100%。与滚动进度时间轴不同，您不能指定滚动条，因为主体的可见性始终在其最近的祖先滚动条中进行跟踪。可以通过两种方式指定提供视图进度时间线的主题：
    • 命名视图进度时间线是使用 view-timeline-name 属性（或 view-timeline 速记属性）显式命名主题的时间线。然后，通过将该名称指定为该元素animation-timeline属性的值，将该名称链接到要进行动画处理的元素。这是一个关键点 - 使用命名视图进度时间线，要制作动画的元素不必与主题相同。
    • 名视图进度时间线是将 view（） 函数作为animation-timeline值，使其根据其在最近的父滚动条中的位置进行动画处理。

设置命名滚动进度时间线 ​

html
<div id="container">
  <div id="square"></div>
  <div id="stretcher"></div>
</div>
css
#container {
    height: 300px;
    overflow-y: scroll;
    scroll-timeline-name: --squareTimeline;
    position: relative;
}
#square {
    background-color: deeppink;
    width: 100px;
    height: 100px;
    margin-top: 100px;
    animation-name: rotateAnimation;
    animation-duration: 1ms; /* Firefox requires this to apply the animation */
    animation-direction: alternate;
    animation-timeline: --squareTimeline;

    position: absolute;
    bottom: 0;
}

@keyframes rotateAnimation {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(360deg);
    }
}
#stretcher {
    height: 600px;
}

设置匿名滚动进度时间线 ​

此特定示例中的时间线由具有（任何）滚动条的最近父元素提供，从块方向的滚动条开始

html
<main>
  <section>
    <article>
     ...
    </article>
  </section>
  <div id="progress"></div>
</main>
css
main {
    height: 200px;
    overflow-y: auto;
}
#progress{
    width: 100%;
    height: 10px;
    background-color: #c00bc0;
    position: fixed;
    top: 0;
    left: 0;

    transform-origin: 0 50%;
    animation: progress linear;
    /* 简写，默认参数是 animation-timeline: scroll(block nearest); */
    animation-timeline: scroll(); 
}
@keyframes progress {
    from {
        transform: translateX(-100%);
    }
    to {
        transform: translateX(0);
    }
}

设置命名视图进度时间线 ​

命名的视图进度时间线是使用class为animation元素上的view-timeline-name属性定义的值为--subjectReveal。 然后使用animation-timeline: --subjectReveal; 将其设置为同一元素的时间轴。结果是，主题元素在滚动文档时向上移动时会进行动画处理。
