❑ Renderer Module

编译器将html转为 render function
响应对象使用响应式模块
渲染模块进入渲染阶段，调用render函数（该render引用了响应对象）
响应对象的变化，render函数返回一个虚拟DOM节点
在Mount挂载阶段，使用虚拟DOM节点创建web页面
响应对象再次变化，渲染器再次调用render函数，创建一个新的虚拟DOM节点，新旧节点发送到补丁函数（patch function）中，根据需要更新网页

virtual-scroll ​

INFO

应用背景是前端接收大量数据，一次性全部渲染所有节点容易造成浏览器卡死，影响体验。 参考vue-virtual-scroll实现思路是仅渲染当前视图可见部分的数据，并在初始化计算所有元素的累加高度，在最外层盒子出现滚动条。 在滚动的过程中，实时刷新渲染对应高度的数据，以此达到减少渲染开支的目的。

html
<div class="content">
  <div class="box"></div>
</div>
css
* {
    padding: 0;
    margin: 0;
}
.content {
    width: 100%;
    height: 200px;
    overflow-y: auto;
}
.box {
    width: 90%;
    overflow: hidden;
    border: 1px solid red;
}
.test-item {
    width: 100%;
    height: 40px;
    text-align: center;
    color: deeppink;
    line-height: 40px;
    background: linear-gradient(#0df6ec, #00bebe);
}
js
const boxEl = document.querySelector('.box')
const contentEl = document.querySelector('.content')
const listLength = 1000 // 样例数
let list // 存储根据样例数生成的节点

const initHeight = () => {
    list = Array.from({ length: listLength }).map(item => ({}))
    list.forEach((item, index) => {
        const divEl = document.createElement('div')
        divEl.classList.add('test-item')
        divEl.id = `id-${index}`
        divEl.innerText = index
        item['el'] = divEl
        item['position'] = 40 * index
    })
    boxEl.style.height = `${listLength * 40}px`
}
// 初始化
initHeight() 
const initEl = document.createElement('div')
initEl.append(...list.slice(0, 10).map(item => item.el))
boxEl.innerHTML = initEl.innerHTML
// 监听滚动事件
contentEl.addEventListener('scroll', e => {
    const scrollTop = contentEl.scrollTop
    let h
    let a = 0
    let b = listLength - 1
    let i = ~~(listLength / 2)
    let oldI

    // Searching for startIndex
    do {
        oldI = i
        h = list[i].position
        if (h < scrollTop) {
            a = i
        } else if (i < listLength - 1 && list[i + 1].position > scrollTop) {
            b = i
        }
        i = ~~((a + b) / 2)
    } while (i !== oldI) 

    const temEl = document.createElement('div')
    const copy = list.slice(i, i+10)
    copy.forEach(item => {
        item['el'].style.transform = `translateY(${40 * i}px)`
    })
    temEl.append(...copy.map(item => item.el))
    boxEl.innerHTML = temEl.innerHTML
})

TIP

重点在do...while循环中，使用二分法查找已滚动距离对应的元素索引
